# Horizon Salon

**Best for:** Hair salons, barbers, beauty studios

## What makes it different:

- Appointment-first layout with big service cards
- Embedded booking section and stylist spotlight areas
- Service menus built to convert walk-ins into bookings

**Style:** Soft, clean, airy layout with rounded cards and warm accents

## Features

- **Appointment-first hero** with prominent "Book Now" CTA
- **Stylist spotlight cards** with images and bios
- **Service menu** with price list and add-to-quote toggles
- **Testimonials carousel** with automatic rotation
- **Map + hours block** with contact information
- **Mobile-first booking modal** for seamless appointment booking
- **Sticky booking CTA** that appears on scroll
- **Responsive design** optimized for all devices
- **Smooth animations** and transitions throughout
- **Soft neutral backgrounds** with warm yellow accent buttons

## Tech Stack

- React 18
- Vite
- TailwindCSS
- Lucide React (icons)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Customization

- Update stylist information in `src/data.js`
- Modify services and pricing in `src/data.js`
- Adjust colors in `tailwind.config.js` (accent-yellow)
- Replace placeholder images with your own
- Update contact information in `MapHours.jsx`
- Customize the Google Maps embed URL in `MapHours.jsx`
